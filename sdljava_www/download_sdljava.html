Dependencies (minimum versions, higher should be fine)

Please note these dependencies apply to the latest released version. If you are working from the CVS version please see the INSTALL file in the source tree.
Required:
  • Java 1.4
  • SDL 1.2.7
Optional:
  • GLEW 1.3.1
    ( OpenGL support)
  • FTGL 2.1.3
    ( Font rending with OpenGL)
  • SDL_image 1.2.3
  • SDL_mixer 1.2.5
  • SDL_ttf 2.0.6
  • SDL_gfx 2.0.13
  • Ant 1.6.2
    ( only if you want to compile java sources)
  • Ruby 1.8.2
    ( only if you want to re-generate native layer)
  • SWIG 1.3.22
    ( only if you want to re-generate native layer)

CVS

Anonymous CVS Access

This project's SourceForge.net CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. The module you wish to check out must be specified as the modulename. When prompted for a password for anonymous, simply press the Enter key.

The module name is "sdljava"

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/sdljava login

cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/sdljava co sdljava

Information about accessing this CVS repository may be found in our document titled, "Basic Introduction to CVS and SourceForge.net (SF.net) Project CVS Services".

Updates from within the module's directory do not need the -d parameter.

NOTE: UNIX file and directory names are case sensitive. The path to the project CVSROOT must be specified using lowercase characters (i.e. /cvsroot/sdljava)

Don't forget to download the testdata if you want to run the image or mixer tests.
